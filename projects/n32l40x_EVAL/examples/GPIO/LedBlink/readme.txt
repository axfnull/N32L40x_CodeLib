1、功能说明

	1、此例程展示读取端口高低电平，控制 LED（D1、D2、D3）闪烁
	2、控制 IO 口电平翻转

2、使用环境

	/* 硬件环境：工程对应的开发硬件平台 */
	开发板：N32L43XM-STB N32L40XM-STB V1.1

3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
	1、SystemClock：64MHz
	2、GPIO：PA8、PB4、PB5 控制 LED(D1、D2、D3) 闪烁


	/* 描述Demo的测试步骤和现象 */
	1.编译后下载程序复位运行；
	2.上电后 D1 常亮，D2、D3 闪烁；
