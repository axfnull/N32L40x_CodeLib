1、功能说明

	/* 简单描述工程功能 */
        这个例程配置并演示CAN1在环回测试模式下收发CAN报文情况


2、使用环境

        软件开发环境：
        IDE工具：KEIL MDK-ARM 5.26
    
        硬件环境：
        开发板 N32L40XM-STB


3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
        SystemClock：64MHz
        USART：TX - PA9，RX - PA10，波特率115200
	CAN1：RX - PA11，TX - PA12，波特率500K，环回测试模式

	/* 描述Demo的测试步骤和现象 */
        1.编译后下载程序复位运行；
        2.查看串口打印信息，CAN1把发送的报文当做接收的报文并保存，当查看到打印信息为接收通过时程序运行正常；
